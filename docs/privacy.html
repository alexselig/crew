Privacy

Privacy, by design.

Last updated: July 23, 2026
The short version: Crew runs entirely on your computer. It has no servers, no accounts, no telemetry, and no analytics — it never sends your data to us or anyone else. Everything below just explains that in detail.

What we collect

Nothing. Crew does not collect, transmit, store, or sell any personal information. There is no sign-up or account, no usage tracking, no analytics, and no crash reporting. Crew has no backend server of its own — there is nowhere for your data to go.

Where your data lives

Everything Crew works with stays on your device:

  • Your sessions and their terminal input and output
  • Transcripts you choose to save
  • Settings, workspaces, and character/color assignments
  • Any API keys or credentials used by the command-line tools you launch

This data is stored locally — in your home folder and the app's support directory — and is never uploaded by Crew.

No telemetry, no phone-home

Crew contains no analytics or telemetry code, and does not automatically update or contact us to check for new versions. The Activity & spend view inside Crew is computed locally from your own sessions — it is not sent anywhere.

The AI tools you run through Crew

Crew is a launcher: it opens third-party command-line agents — such as Claude Code (Anthropic) and GitHub Copilot CLI — inside terminals it manages. When you use those tools, they connect to their providers' services and handle your prompts and code according to their own policies. Crew does not control, intercept, or receive that traffic.

Please review each tool's own policy, for example the Anthropic Privacy Policy and the GitHub Privacy Statement.

This website

crewcli.dev is a static site hosted on GitHub Pages. It sets no cookies and runs no analytics or trackers. As with any website, the host (GitHub) may process basic technical information such as your IP address and request logs in order to serve the page and downloads; that processing is covered by the GitHub Privacy Statement. Application downloads are served from GitHub Releases.

Changes to this statement

If this statement ever changes, the updated version will be posted on this page with a new date above.

Contact

Questions about privacy? Open an issue at github.com/alexselig/crew/issues.
